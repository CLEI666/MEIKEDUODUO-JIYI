## 一、选品标准

### 核心要求：客单价小、利润高、小件产品

1. **不做食品** - 七戒明确不做食品类
2. **不做手机** - 七戒明确不做手机产品
3. **不做化妆品** - 七戒明确不做化妆品（护肤品、美妆、精油等）
4. **采购价低** - 1688采购价建议≤50元
4. **利润率高** - 目标利润率≥20%
5. **小件产品** - 体积小、重量轻，降低头程运费
6. **客单价小** - 美客多售价≤500比索（≈175元人民币）

### ★★★ 强烈推荐

## 三、利润计算公式

```
预估利润 = 售价 - 采购成本 - 头程运费 - 平台佣金 - 广告费 - 退货损耗
利润率 = (预估利润 / 售价) × 100%
```

**默认参数：**
- 平台佣金：15%-20%
- 广告费：10%
- 退货损耗：5%-10%
- 头程运费：货值×0.3-0.5（经验值）

---

## 四、1688采购注意事项

1. 选择回头率≥40%的供应商
2. 起订量（MOQ）越低越好
3. 优先选择有厂家的供应商
4. 注意产品是否有品牌侵权风险

## 五、每日简报标准（9项必须包含）

1. **供应商联系方式** - ⭐必须包含：1688店名、旺旺/微信等联系方式、起订量、回头率
2. **产品图片** - 实物图+白底图 ⭐必须带图片
3. **进货价格** - 1688最低报价
4. **平台销售价格** - 美客多当前最低价/中位价
5. **成本核算** - 头程运费 + 平台佣金15%-20% + 广告费10% + 退货损耗5%-10%
6. **利润估算** - 利润率 + 预估利润（人民币结算）
7. **竞品分析** - 排名前3的竞品价格和评分
8. **选品结论** - 强烈推荐/潜力/风险
9. **爆品预测** - 基于热销榜和趋势，预测未来3个月爆品，单独列出
