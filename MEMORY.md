### 利润计算公式
```
预估利润 = 售价 - 采购成本 - 头程运费 - 平台佣金 - 广告费 - 退货损耗
利润率 = (预估利润 / 售价) × 100%
```
**默认参数：**
- 佣金：15%-20%
- 广告费：10%
- 退货损耗：5%-10%
